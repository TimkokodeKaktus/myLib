=============================================
    Readme for myLib
=============================================
File structure "users.txt" :

"name-surname-"

name: user name
surname: user surname
----------------------------------------------
File structure "books.txt":

"title-status-authorname-authorsurname-"

title: book title
status: book status
satus == 1  in library; status == 0 on reader

----------------------------------------------
File structure "orders.txt"

"idBook-idUser-status-"

idBook - registration number book in library
idUser - registration number reader in library
status - order status
status == 0 - borrowed; status == 1 - returned

----------------------------------------------
